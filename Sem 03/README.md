## Итерационни изрази

### While цикъл
Това е най-простият цикъл и неговият синтаксис е  

```c++
while(<condition>){
//statement
}
```
Т.е. тялото на цикъла ще се изпълнява докато <condition> е вярно, или като flow-chart

### Пример

  while(n>0){

### Пример с алгоритъма на Евклид  

```c++
	n = 16, k = 24
	<24, 16>  24 % 16 = 8
	<16, 8>  16 % 8 = 0
	<8, 0> 
	=> gcd(16,24) = 8
```  
или като код

```c++

	int n, k;
	std::cin >> n >> k;
	// <n,k> 

	if (k > n) { //swap
		int temp = k;
		k = n;
		n = temp;
	}

	// n>k -> n = k*r + q
	while (k != 0) {
		int mod = n % k;
		n = k;
		k = mod;
	}

	std::cout << n;
```  

## Do-While цикъл
Аналогичен на While цикъла с единственото изключение, че първо се изпълнява тялото, а след това се прави проверка, т.е. по този начин се обезпечава поне една реализация на тялото  

```c++
do{
    //statement
}while(<condition>)  
```

## For цикъл
  
```c++
for(initialization; condition; increase){
  //code to be executed
} 
```
Както и при while, for цикъла повтаря //statement докато condition е вярно. В допълнение, for цикълът предлага конкретни места, които съдържат изрази за инициализация и нарастване, които се изпълняват преди и след всяка итерация, респективно. Той работи по следния начин  
  1. Изпълнява се initialization. Обикновено, тук се дефинира брояч и му задава първоначална стойност. Това се изпълнява само веднъж - в началото на цикъла;
  1. Проверява се condition. Ако condition е вярно, то се изпълнява тялото на цикъла. В противен случай, тялото не се изпълнява и се излиза извън цикъла;
  1. Изпълнява се тялото на цикъла - както обикновено там може да имаме един единствен израз, или блок от команди, ограден с {};
  1. Изпълнява се increase и се връщаме при стъпка 2;
 
Например  
```c++
for(int n=10; n>0; --n){
  std::cout<<n<<", ";
} 
```

## Jump statements
Тези изрази позволяват промяна на flow-a на една програма:  
  - break - излиза/терминира от цикъла, даже и <condition> да е вярно (при цикъла);  
  - continue - налага пропускане на последващата част от тялото на цикъла, като преминава на следващата итерация;
  - goto - предлага преминаване до изцяло друга част на програмата, като частта се указва с label. Този скок игнорира целия естествен flow, по който е построена програмата. Следователно, (препоръчително е да не) се използва с огромно внимание. Например,  
  
```c++
int n;
mylabel:     
		
std::cin>> n;		
if(n == 3) goto mylabel;   //jumps to mylabel if the value of n is equal to 3
		
std::cout<<n;

```
